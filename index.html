FAQ

Got Questions?

Who is this workshop designed for? +
This workshop is tailored for developers, UI engineers, and technical designers who already possess solid knowledge of HTML, CSS, and basic JS, and want to level up their skills in creative coding, particle systems, layout animation APIs, and performant GPU-driven web graphics.
What technologies will we be using? +
We will focus purely on native web technologies: HTML5 Canvas, WebGL, GLSL Shaders, Modern CSS (Grid, Flexbox, custom properties), and vanilla JavaScript. No complex frameworks like React or Three.js are required, as we build from first principles.
Are the live streams recorded for later viewing? +
Yes! All live sessions, coding labs, and feedback panels will be recorded in 4K and made available to registered students, alongside source templates, resources, and exclusive community chat channels for lifetime access.
What are the hardware prerequisites? +
Any modern laptop or desktop (macOS, Windows, or Linux) with a web browser (Chrome, Firefox, or Safari) and a code editor installed (VS Code is recommended). Since we run GPU code, a dedicated graphics card is helpful but not strictly necessary.
